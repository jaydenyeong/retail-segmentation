# Customer Segmentation with RFM & K-Means (Power BI)

## Overview
This project analyzes retail reansaction data to segment customers using RFM analysis and K-means clustering. THe goal is to support data-driven marketing, retention, and revemue optimization

## Tools
- Power BI
- Python (pandas, scikit-learn)
- RFM Analysis
- K-means clustering

## Dashboard Pages
1. Executive overview
2. RFM Segmentation
3. K-means Clusters
4. RFM vs K-means validation & Customer Drill-Down

## Key Insights
- High0value customers (CHampions) contribution a disproportionate share of revenue
- Strong alignment between RFM and K-means validates segmentation quality
- K-means refines mid-value segments for precision targeting
